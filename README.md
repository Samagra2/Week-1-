# Week 1: Python Basics – Personal Introduction Program

## Project Overview
This project is designed to help beginners understand the fundamentals of Python
programming by creating a simple interactive program that introduces a user.

## Objectives
- Learn Python syntax and structure
- Understand variables and data types
- Take user input using input()
- Display output using print() and f-strings

## Tools Used
- Python 3
- Google Colab

## Setup Instructions
1. Open Google Colab
2. Copy the code from `personal_intro.py`
3. Paste it into a new Colab notebook or Python file
4. Run the program and provide input when prompted

## Project Files
- `personal_intro.py` – Main Python program
- `requirements.txt` – Python version requirement
- `screenshot.png` – Output screenshot proof

## Learning Outcomes
- Basics of Python programming
- User interaction through input/output
- Writing clean beginner-level code

## Testing
The program was tested using different names, ages, hobbies, and cities to ensure
correct and friendly output.
